### 1-3. 상품 카드 구성 요소

```
┌──────────┐
│  상품    │  ← 세로형 이미지 (9:16 비율 추정)
│  이미지  │
│        ♡ │  ← 찜 버튼 (우상단 고정)
├──────────┤
│ 마켓명   │  ← 소문자 텍스트
│ 상품명   │
│ ₩가격    │  ← 무료배송 뱃지 병기
└──────────┘
```

- 뱃지: 무료배송 (전 상품 정책 반영), 할인율 **미확인**
