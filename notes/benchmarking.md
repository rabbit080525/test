## 1. 무신사

### 1-1. 피드 철학
멀티스토어 구조로 전면 개편 (2024년 하반기, "무신사 2.0").
단일 홈이 아닌 **10개의 전문 탭(판)** 으로 쇼핑 목적을 분산.
컴포넌트를 모듈화해 어떤 탭에서도 일관된 UX를 유지.

> 출처: [[팀무신사] 새로워진 무신사 UI·UX — 무신사 뉴스룸](https://newsroom.musinsa.com/newsroom-menu/2024-0920-teammusinsa) [신뢰도: A]

| 탭(판) | 주요 콘텐츠 |

| **랭킹판** | 30분 단위 실시간 랭킹. 상품·브랜드·검색어·스타일별(20개+) 랭킹 |
| **업데이트판** | 신상품 발매, 재발매, 기획전, 할인, 판매 예정 상품 |
| **브랜드판** | 브랜드 이슈·위크, 신규·인기 브랜드, 디자이너, 브랜드 동향 |
| **스타일판** | 패션 에디터 스타일링 코디, 패션 피플 코디 콘텐츠 |

### 1-3. 주요 UI 컴포넌트

**상품 카드**
- 상품 이미지 (정방형/세로형 혼용)
- 브랜드명 + 상품명 + 가격
- 좋아요(찜) 버튼 우상단 고정
- 배지: NEW / BEST / 단독 / 한정 / 세일율

**개인화 추천 영역**
- "추천판": 탐색 상품과 연관·유사 상품 AI 추천
- 개인화 추천 영역 구매 전환율 **3배 이상** 향상 (공식 발표)
- 상세 페이지 조회수 **2배 이상** 상승 (공식 발표)

**내비게이션**
- 상단: 검색바 + 장바구니 + 알림 아이콘
- 하단: 홈 / 탐색 / 스타일 / MY 탭
- 홈 내부: 10개 판 수평 스크롤 탭

**디자인 시스템**
- 그레이 배경 + 화이트 카드 컴포넌트
- 모듈 단위 설계로 탭마다 재사용
- 유연성·확장성 중심 컴포넌트 패턴

> 출처: [무신사 UI 개편: 복잡함을 간결하게 — 디자인 나침반](https://designcompass.org/2024/08/26/musinsa-ui-update/) [신뢰도: B]
> 출처: [UX/UI 개편한 무신사, 구매 전환율 3배 ↑ — 패션비즈](https://fashionbiz.co.kr/article/212657) [신뢰도: B]

### 1-4. 개인화 전략 특이점
- 전체 피드보다 **탭(판) 단위 개인화** 전략 채택
- 추천판에서만 AI 개인화, 나머지 탭은 전체 공통 콘텐츠
- **추정**: 탭 단위 분리는 개인화와 탐색(Discovery)의 균형 전략으로 보임

## 2. 에이블리

### 2-1. 피드 철학
"초개인화 AI 쇼핑" 슬로건.
앱 진입 순간부터 **모든 화면이 사용자별로 다르게** 구성.
취향 데이터를 플랫폼의 핵심 자산으로 정의.

> 출처: [취향 데이터 25억개 확보한 에이블리 — 유니콘팩토리](https://www.unicornfactory.co.kr/article/2024021917324839859) [신뢰도: B]
> 출처: [에이블리 — Google Play 앱 소개](https://play.google.com/store/apps/details?id=com.banhala.android&hl=en_US) [신뢰도: A]

---

### 2-2. 홈 피드 구조

**데이터 규모**
- 누적 행동 데이터: **1,500억 건 이상**
- 구매 의향 데이터(찜·리뷰·장바구니): **35억 건 이상**
- 하루 **4억 건** 빅데이터 실시간 축적

> 출처: [에이블리, 하루 4억 건 빅데이터 축적 — VentureSquare](https://www.venturesquare.net/1040139) [신뢰도: B]

**탭 구성**
- **BEST 탭**: 인기 상품 랭킹
- **NEW 탭**: 신상품 피드
- **취향 피드**: 개인 맞춤 상품 무한 스크롤 (핵심 탭)
- **추정**: 카테고리 필터(상의/하의/원피스 등) 수평 스크롤 제공

**개인화 메커니즘**
- 상품 찜, 리뷰, 체형 데이터 → 개인 맞춤 추천
- AI 알고리즘으로 **구매 전환율 4배** 향상
- 2026년 목표: 스타일 데이터 + 카테고리 교차 추천으로 "한국인의 취향 지도" 제작

> 출처: [에이블리, AI 추천 기술 강화 — 유니콘팩토리](https://www.unicornfactory.co.kr/article/2024021917324839859) [신뢰도: B]

### 2-3. 주요 UI 컴포넌트

**상품 카드**
- 세로형 이미지 중심 (9:16 비율 추정)
- 마켓명 + 상품명 + 가격
- 무료배송 뱃지 (전 상품 무료배송 정책 반영)
- 찜 버튼 상단 고정

**피드 구성**
- 2열 그리드 기본 (상품 카드 밀도 높음)
- 홈 최상단: 광고 배너 (경쟁사 대비 높이 큼)
- "사용자를 위한 추천 상품" 섹션 (AI 개인화)
- **추정**: 광고 배너 높이가 커 초기 피드 노출 면적이 경쟁사보다 적음

**내비게이션**
- 하단: 홈 / 카테고리 / 찜 / MY
- 카테고리 필터: 수평 스크롤 칩 형태

**개인화 UX**
- 앱 진입 시 사용자별 다른 홈 화면 노출
- 필터 기능으로 세부 취향 조정 가능

### 2-4. 개인화 전략 특이점
- 에이블리 → 지그재그 대비 추천 종류가 단순한 편 (과거 분석 기준)
- **추정**: 최근 1,500억 건 데이터 활용으로 추천 다양성 개선 중
- 향후 라이프스타일·남성(4910) 카테고리 교차 추천으로 확장 예정

### 3-2. 홈 피드 구조

**탭 구성**

### 3-3. 주요 UI 컴포넌트

### 3-4. 개인화 전략 특이점
- 알고리즘 개인화보다 **MD 큐레이션 신뢰**를 핵심 가치로 설정
