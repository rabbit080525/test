# 개인화 피드 추천 알고리즘 & AI 기술

> 작성일: 2026-04-10

---

## 1. 주요 알고리즘 유형

### 1-1. 협업 필터링(Collaborative Filtering)
- **개념**: 비슷한 취향의 다른 사용자들이 좋아한 상품을 추천
- **User-based**: 유사 사용자 그룹의 구매 패턴 활용
- **Item-based**: 함께 구매되는 상품 간의 유사성 활용
- **활용 사례**: 쿠팡 — 협업 필터링 + 딥러닝 결합으로 개인화 정확도 40% 향상

### 1-2. 콘텐츠 기반 필터링(Content-Based Filtering)
- **개념**: 사용자가 이전에 본/구매한 상품의 속성(카테고리, 브랜드, 가격대, 색상 등)과 유사한 상품 추천
- 버티컬 이커머스에 특히 효과적 (카테고리 속성이 정교함)
- 신규 사용자보다 **재방문 사용자**에게 강점

### 1-3. 하이브리드 모델
- 협업 필터링 + 콘텐츠 기반 + 딥러닝 조합
- 콜드 스타트(신규 사용자) 문제를 콘텐츠 기반으로 보완
- 2025년 기준 대부분의 대형 플랫폼이 하이브리드 채택

### 1-4. 딥러닝 기반 추천
- **순환신경망(RNN/LSTM)**: 사용자의 세션 내 행동 시퀀스 학습
- **Transformer**: 긴 행동 이력에서 패턴 추출 (BERT4Rec 등)
- **Graph Attention Networks(GAT)**: 사용자-상품 간 소셜 연결 관계 반영 (2025년 학술 트렌드)

### 1-5. 실시간 맥락 추천(Contextual Bandits)
- 시간대, 디바이스, 날씨, 위치 등 **현재 맥락**을 실시간 반영
- 쿠팡: 시간대·지역별 구매 패턴 학습 → 지역 특화 추천으로 전환율 25% 이상 향상
- A/B 테스트를 자동화해 최적 추천 전략 선택

## 3. 데이터 수집 및 처리

### 수집 데이터 유형
| 유형 | 예시 |
|------|------|
| 명시적 데이터 | 평점, 위시리스트, 리뷰 |
| 암묵적 데이터 | 클릭, 스크롤 깊이, 체류 시간, 장바구니 추가/삭제 |
| 맥락 데이터 | 시간대, 기기, 위치, 날씨, 이전 세션 |
| 소셜 데이터 | 공유, 팔로우, UGC 참여 |

### 처리 파이프라인 (일반적)
```
원시 데이터 수집
  → 특성 엔지니어링 (Feature Engineering)
  → 모델 학습 (배치 + 실시간 스트리밍)
  → 추천 서빙 (저지연 캐싱)
  → A/B 테스트 및 피드백 루프
```

## 4. 과개인화(Over-Personalization) 주의사항

- 너무 많은 맞춤 추천은 오히려 사용자 피로 유발
- "필터 버블": 다양성 부족으로 새로운 발견 기회 감소
- **권장**: 개인화 추천 + 트렌딩/편집 큐레이션을 적절히 혼합
- 데이터 활용 정책 투명하게 공개 (GDPR·개인정보보호법 준수)
