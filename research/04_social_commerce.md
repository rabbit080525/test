## 1. 소셜 커머스 시장 현황 (2025–2026)

- 소셜 플랫폼에서 쇼핑이 이루어지는 소셜 커머스가 이커머스의 핵심 채널로 부상
- 발견(Discovery) → 구매(Purchase)가 앱 이탈 없이 단일 플랫폼 내에서 완결
- 전통 이커머스의 피드 UX가 소셜 플랫폼의 알고리즘 피드를 적극 벤치마킹

## 3. TikTok 피드 방식의 이커머스 적용

### 핵심 메커니즘
- **For You Page(FYP)** 방식: 팔로우 관계 무관, 순수 알고리즘 기반 노출
- 시청 완료율, 공유, 댓글, 구매 클릭 등 **참여 시그널**로 피드 최적화
- 첫 3–7초 훅(Hook)이 피드 노출 지속 여부를 결정

### 이커머스 피드에 적용 가능한 원칙
1. **참여 기반 랭킹**: 단순 클릭이 아닌 체류 시간·스크롤 깊이·공유 등 복합 시그널 활용
2. **콘텐츠-상품 혼합**: 상품만 노출하지 않고 스타일링·리뷰 영상 삽입
3. **즉각적 구매 경로**: 피드에서 PDP 없이 바로 구매 가능한 Quick Buy
4. **알고리즘 투명성**: "이 상품을 추천한 이유" 표시로 신뢰도 향상

## 4. 숏폼 비디오 커머스 설계 원칙

### 콘텐츠 유형별 전환 효과
- **Pre-recorded 크리에이터 영상**: TikTok Shop 매출의 약 2/3 기여 (On-demand 선호)
- **라이브 스트리밍**: 실시간 Q&A + 한정 할인 → 즉각적 구매 욕구 자극
- **UGC 리뷰**: 실 구매자 영상 → 상품 신뢰도 상승

### 버티컬 이커머스 적용 체크리스트
- [ ] 세로형(9:16) 영상 우선 지원
- [ ] 영상 내 상품 태그 → 탭 시 빠른 구매 모달
- [ ] 영상 재생 중 스크롤 시 미니 플레이어 유지
- [ ] 영상 완료 후 관련 상품 자동 추천

## 5. 발견형 쇼핑(Discovery Commerce) 설계 전략

### 사용자 인사이트
- 소비자가 **능동 검색하지 않아도** AI가 취향에 맞는 상품을 먼저 제시
- 스크롤하다 '끌림'을 느끼는 순간 즉시 구매 → 충동 구매 패턴

### 피드 구성 전략
```
발견형 피드 = 개인화 추천 + 트렌딩 + 큐레이션 + 소셜 증거

    [개인화 추천 40%]  ← AI 기반, 사용자 취향 최적화
    [트렌딩 30%]       ← 실시간 인기, 신선함 확보
    [편집 큐레이션 20%] ← 전문가·MD 큐레이션, 브랜드 아이덴티티
    [소셜 증거 10%]     ← "친구 OO명이 구매", 리뷰 하이라이트
```
