# 버티컬 이커머스 개인화 피드 페이지 — 리서치 개요

> 작성일: 2026-04-10  
> 목적: 버티컬 이커머스 쇼핑몰 내 개인화 피드 페이지 기획 및 설계를 위한 시장 조사

---

## 1. 배경 및 정의

### 버티컬 이커머스란?
패션, 뷰티, 식품, 스포츠 등 특정 카테고리에 집중한 전문 쇼핑몰.  
수평(종합) 이커머스(쿠팡, 11번가)와 달리 **깊은 카테고리 전문성**과 **타깃 고객 집중**이 핵심 경쟁력.

### 개인화 피드란?
사용자의 행동 데이터(클릭, 구매, 체류시간, 검색)를 AI/알고리즘으로 분석해  
**각 사용자에게 최적화된 상품/콘텐츠 스트림을 자동으로 구성**하는 페이지.

---

## 2. 2025–2026 핵심 트렌드

### 2-1. 발견형 쇼핑(Discovery Commerce)의 부상
- 기존 목적형 쇼핑(검색 → 구매)에서 **발견형 쇼핑**으로 패러다임 전환
- 소비자가 검색하지 않아도 AI가 취향에 맞는 상품을 먼저 제시
- TikTok·Instagram Reels의 For You Feed 방식이 이커머스로 이식

### 2-2. AI 기반 동적 인터페이스
- 피드 레이아웃, 콘텐츠 순서, 배너 등이 **사용자별 실시간 변동**
- 반복 방문자에게는 이전 구매/탐색 패턴 기반의 맞춤 화면 제공
- 추천 → 상품 묶음(번들) 생성 → 대화형 쇼핑 어시스턴트로 확장

### 2-3. 콘텐츠-커머스 통합
- 숏폼 비디오 + 상품 태그 → 스크롤 중 즉시 구매
- UGC(User Generated Content)가 피드 내 상품 신뢰도의 핵심 요소
- TikTok Shop 전환율 4.7% (Instagram Shopping 2.1%의 약 2배)

### 2-4. 모바일 퍼스트, 버티컬 스크롤
- 2025년 전 세계 온라인 쇼핑객의 91%가 스마트폰으로 구매
- 세로 스크롤 + 스티키 "장바구니 담기" 버튼이 표준 UX
- 데스크탑에도 모바일 스타일의 롱 스크롤 레이아웃 적용 확대

### 2-5. 자연스러운 개인화
- 팝업·강제 알림 방식 탈피 → 피드 자체에 개인화가 내재
- 맥락 기반 추천(시간대·날씨·지역·이전 구매) 정교화
- 데이터 프라이버시 동의 기반의 투명한 개인화 운영

## 3. 주요 글로벌 레퍼런스

| 플랫폼 | 특징 |
|--------|------|
| TikTok Shop | 숏폼 비디오 + 인앱 구매, For You 피드 기반, CR 4.7% |
| Pinterest | 무한 스크롤 그리드, 관심사 기반 발견, 비주얼 검색 |
| Instagram Shopping | 릴스 + 상품 태그, 팔로우 기반 + 알고리즘 혼합 피드 |
| Amazon | 협업 필터링 + 딥러닝, 홈 피드 완전 개인화 |
| 쿠팡 | 협업 필터링 + 딥러닝 결합, 지역·시간대 맥락 추천, CR +25% |
