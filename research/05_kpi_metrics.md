# 개인화 피드 KPI & 성과 지표

> 작성일: 2026-04-10

---

## 1. 핵심 KPI 체계

### 1-1. 비즈니스 핵심 지표

| KPI | 설명 | 벤치마크 |
|-----|------|---------|
| 구매 전환율(CVR) | 피드 방문자 중 구매 완료 비율 | 이커머스 평균 1–3%, 개인화 적용 시 3–5% 목표 |
| 평균 주문액(AOV) | 1회 구매 평균 금액 | 개인화 추천 세션: 비추천 대비 369% 상승 사례 |
| 고객생애가치(LTV) | 고객 전체 기간 누적 구매액 | 장기 성과 지표, 재구매율과 연동 |
| 고객획득비용(CAC) | 신규 고객 1명 확보 비용 | LTV/CAC > 3 목표 |
| 추천 기여 매출 | 피드 추천을 통해 발생한 매출 비율 | Amazon 기준 35% |

### 1-2. 피드 참여 지표

| KPI | 설명 | 목표 |
|-----|------|------|
| 피드 체류 시간 | 개인화 피드 페이지 평균 체류 시간 | 3분 이상 목표 |
| 스크롤 깊이 | 피드에서 얼마나 아래까지 스크롤하는가 | 75% 이상 도달 비율 |
| CTR(클릭률) | 노출된 상품 카드 클릭 비율 | 5–10% (카테고리별 상이) |
| 위시리스트 추가율 | 피드 상품 중 위시리스트 저장 비율 | 이후 구매 전환 추적 |
| 피드 재방문율 | 피드 방문 후 재방문 비율 | 30일 내 재방문 50% 이상 |

### 1-3. 추천 품질 지표

| KPI | 설명 |
|-----|------|
| 추천 클릭률(Rec CTR) | 추천 상품 카드 클릭 비율 |
| 추천 전환율(Rec CVR) | 추천 상품 클릭 후 구매 전환율 |
| 추천 다양성(Diversity) | 동일 카테고리·브랜드 반복 추천 비율 (낮을수록 좋음) |
| 신선도(Freshness) | 최신 상품·트렌드 피드 반영 속도 |
| 콜드 스타트 성능 | 신규 사용자 첫 3회 세션 내 전환율 |

## 2. 피드 성과 측정 방법론

### A/B 테스트 프레임워크
```
피드 개선 사이클
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  가설 수립
    → 실험 설계 (대조군/실험군 분리)
    → 실행 (최소 1–2주)
    → 결과 분석 (통계적 유의성 확인)
    → 승자 롤아웃 or 추가 반복
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 측정 포인트별 트래킹 이벤트
| 위치 | 트래킹 이벤트 |
|------|-------------|
| 피드 진입 | page_view, feed_load |
| 상품 노출 | product_impression (뷰포트 50% 이상) |
| 상품 클릭 | product_click + 추천 위치(position) |
| 장바구니 추가 | add_to_cart + 추천 여부 flag |
| 구매 완료 | purchase + 추천 기여 여부 |
| 피드 이탈 | scroll_depth, session_end |

## 3. 개인화 수준별 기대 효과

| 개인화 수준 | 설명 | 예상 CVR 향상 |
|------------|------|-------------|
| 0단계 (없음) | 전체 동일 피드 | 기준 |
| 1단계 (룰 기반) | 카테고리·성별·연령대 세그먼트 | +10–20% |
| 2단계 (ML 기반) | 개인 행동 이력 기반 추천 | +30–50% |
| 3단계 (실시간 맥락) | 현재 세션 + 맥락 반영 | +50–80% |
| 4단계 (생성형 AI) | 대화형·동적 인터페이스 적용 | +80%+ (초기 단계) |

## 4. 이커머스 업계 평균 벤치마크

| 지표 | 평균 | 상위 20% |
|------|------|---------|
| 이커머스 평균 CVR | 1–2% | 3–5% |
| TikTok Shop CVR | 4.7% | — |
| 개인화 추천 AOV 증가 | +15–25% | +50%+ |
| 추천 엔진 매출 기여 | 20–35% | 35%+ |
| 재방문 고객 CVR | 비방문 대비 3–5배 | — |
